# 사운드 파일 안내

이 폴더에 아래 파일을 넣으면 게임에서 자동으로 재생됩니다.
(파일이 없어도 게임은 정상적으로 실행되며, 사운드만 재생되지 않습니다.)

- `bgm.wav` : 홈 화면 진입 시 자동 재생되는 배경음악 (반복 재생)
- `click.wav` : 버튼 클릭 등 효과음 (예: 설정 버튼)

## 형식 안내
- `javax.sound.sampled` API를 사용하므로 **WAV(PCM)** 형식을 권장합니다.
- MP3는 기본 자바 사운드 API로 바로 재생되지 않습니다. WAV로 변환해서 사용해주세요.

## 사운드/효과음 추가 방법
1. `sounds/bgm.wav` 파일을 이 폴더에 넣습니다.
2. 다른 효과음을 추가하고 싶다면 `SoundManager.playSfx("파일명.wav")` 형태로
   원하는 코드 위치(카드 공개, 베팅 버튼 클릭 등)에 한 줄만 추가하면 됩니다.

   예시:
   ```java
   SoundManager.playSfx("card.wav"); // sounds/card.wav 재생
   ```

설정(⚙️) 화면에서 배경음악/효과음을 각각 켜고 끄거나 볼륨을 조절할 수 있으며,
설정값은 `settings.properties` 파일에 자동 저장되어 다음 실행 시에도 유지됩니다.
